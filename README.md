# YouTube Homepage Clone 🎬

This is a responsive clone of the YouTube homepage built using **HTML** and **CSS**.

## 🔧 Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)

## 📱 Features
- Fully responsive layout
- Video grid section
- Sidebar and fixed header
- Clean, semantic code
